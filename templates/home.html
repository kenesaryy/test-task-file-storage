API Endpoints
  • POST /api/storage/upload/ - Загрузка файла
  • GET /api/storage/files/ - Список файлов
  • GET /api/storage/download/{id}/ - Скачать файл
  • DELETE /api/storage/delete/{id}/ - Удалить файл
  • GET /api/storage/logs/ - Просмотр логов
Особенности
  • Максимальный размер файла: 16MB
  • Автоматическое разбиение на 16 частей
  • ZIP архивация каждой части
  • Проверка целостности (SHA256)
  • Полное логирование действий
  • Token-based аутентификация
